- Can't "buzz" between two horrors

**The Distinction:**

**Genuine polarity:** Both poles have value, tension is productive
- Example: Security ‚Üî Liberty (both good, navigate between)

**False binary (Sophie's choice):** Both poles unacceptable, tension is trap
- Example: Child A dies ‚Üî Child B dies (both bad, reject frame)

Reject the premise. Refuse the binary. Change the rules.
- Question the framing (Is this really the only choice?)
- Look for hidden third options
- Change the game (What if the rules are wrong?)
- Reject the binary

**When to Dance vs. When to Reprogram:**

**Dance (polarity navigation):**
- Both poles legitimate
- Tension productive
- Movement between poles = wisdom
- Example: Justice ‚Üî Mercy

**Reprogram (reject premise):**
- Both poles unacceptable
- Tension destructive
- Accepting binary = losing
- Example: Impossible dilemmas

**The Discernment:**
1. Do both poles have legitimate value? ‚Üí If yes, probably polarity
2. Are both poles genuinely unacceptable? ‚Üí If yes, probably false binary
3. Does accepting the frame guarantee harm? ‚Üí If yes, question frame
4. Is there creative third option being ignored? ‚Üí If yes, Kobayashi Maru it

**Why This Matters:**

This prevents TensionLines from becoming dogmatic. Acknowledges:
- Not everything is a polarity
- Sometimes you DO need to reject binaries
- The framework has limits
- Discernment matters enormously

"I don't know how to dance between a Sophie's choice."

**Idea #025:** "The Accommodation Visibility Paradox"
- **Quote:** "Why is it when you expose the accommodation it is controversial but when it is hidden no problem? E.g. women's basketball vs men's 3 point line distance controversy problem, size of basketball no problem being different."
- **Tags:** #accommodation #visibility #controversy #equality #equity #dignity
- **Core Insight:** Accommodations that are invisible/normalized are accepted. Same accommodations when made explicit become controversial.
- **Status:** üîµ New - awaiting organization

**The Pattern:**

**Hidden accommodation = No controversy**
- Women's basketball uses smaller ball (28.5" vs 29.5")
- No one objects
- Normalized, invisible

**Exposed accommodation = Controversy**
- Proposal to move women's 3-point line closer
- Immediate controversy
- Made explicit, visible

**Why This Happens:**

1. **Visibility = Vulnerability:** Explicit differences invite scrutiny
2. **Status Quo Bias:** Existing = "natural," new = "special treatment"
3. **Dignity vs. Deficiency:** Explicit accommodation can feel like admission of inadequacy
4. **Equality Fiction:** Hidden allows belief in "same game," explicit forces recognition of difference

**The Paradox:**

**Hidden accommodation:**
- Pro: Avoids stigma, preserves dignity, normalizes difference
- Con: Perpetuates ignorance, prevents optimization, locks in status quo

**Explicit accommodation:**
- Pro: Allows evaluation, transparency, optimization
- Con: Invites controversy, can stigmatize, highlights difference

**Connection to #024 (Agency):**

Hidden accommodations might serve agency better:
- No feeling of "special treatment"
- Preserves dignity of "competing on same terms"

Explicit accommodations risk undermining agency:
- "We're changing rules for you" = implies inadequacy
- Even if intended to help

**The Uncomfortable Truth:**

People are fine with accommodation as long as they don't have to think about it. Making them think about it = controversy, even if accommodation is justified.

**Why It Matters:**

This pattern shows up everywhere:
- Disability accommodations (infrastructure vs. policies)
- Racial equity (implicit bias training vs. explicit quotas)
- Gender differences (some accepted, others debated)

**The Strategic Insight:**

Not all accommodations should be made explicit. Strategic visibility based on what serves agency/dignity best.

**Cross-references:**
- Directly related to #024 (agency): Visibility affects agency differently
- Links to #020-#022: Maybe explicit vs. hidden is false binary‚Äîneed strategic visibility
